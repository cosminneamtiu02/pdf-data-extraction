## Short description

Thin `ExtractionEngine` that constructs LangExtract function call parameters from a `Skill` (prompt, examples, output schema) and invokes LangExtract with the injected `IntelligenceProvider`, returning a `list[RawExtraction]` with per-field character offsets and grounding flags, while containing LangExtract imports to this file.
