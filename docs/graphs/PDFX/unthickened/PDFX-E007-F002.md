## Short description

Start the service successfully even when Ollama is unreachable at boot — the container does not exit, `SkillManifest` validation still runs, `/ready` returns 503, extraction requests return `INTELLIGENCE_UNAVAILABLE`, and the service self-heals as soon as the next successful Ollama probe flips `/ready` green, without requiring a restart.
