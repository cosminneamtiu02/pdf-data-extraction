## Short description

`POST /api/v1/extract` handler: accept multipart/form-data (pdf, skill_name, skill_version, output_mode), reject with `PDF_TOO_LARGE` (413) when byte size exceeds `Settings.max_pdf_bytes` before parsing, call `ExtractionService.extract`, and serialize per output mode — `JSON_ONLY` as `application/json`, `PDF_ONLY` as `application/pdf`, `BOTH` as a hand-rolled `multipart/mixed` response with a JSON part and a PDF part.
