## Short description

Define the `IntelligenceProvider` `typing.Protocol`, the `GenerationResult` dataclass (`data`, `attempts`, `raw_output`), and the provider-agnostic `StructuredOutputValidator` that cleans markdown fences, extracts JSON from prose, validates against the skill's JSONSchema, and retries up to 3 additional times (4 total attempts) on any parse or validation failure.
