## Short description

Concrete provider class that simultaneously satisfies the internal `IntelligenceProvider` protocol and LangExtract's community provider plugin contract, holding a single Ollama HTTP client configured by `Settings.ollama_*` (with the smallest Gemma 4 variant as the default model tag) and containing all Ollama-client imports to this single file.
