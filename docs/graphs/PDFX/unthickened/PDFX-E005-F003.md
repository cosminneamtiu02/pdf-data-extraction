## Short description

Turn each `RawExtraction` into an `ExtractedField` with `BoundingBoxRef`s by looking up start and end blocks in the `OffsetIndex`, dispatching to `SubBlockMatcher` for single-block spans (with whole-block fallback), returning multiple `BoundingBoxRef`s for multi-block and cross-page spans, and returning `grounded=false` with empty `bbox_refs` when no block contains the offsets.
