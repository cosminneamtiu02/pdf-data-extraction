## Short description

Orchestrate the full pipeline in `ExtractionService.extract`: resolve the skill, parse the PDF, concatenate text, invoke the extraction engine, run the span resolver unconditionally, and conditionally invoke the annotator. The whole pipeline runs under `asyncio.timeout(Settings.extraction_timeout_seconds)`, raising `INTELLIGENCE_TIMEOUT` (504) on exceed.
