## Short description

Locate an extracted value inside a block's text via a three-step fallback chain — direct substring, whitespace-normalized substring, Unicode NFKC-normalized substring — returning a `CharRange` on success or `None` on total failure, so `SpanResolver` can compute tight sub-block bounding boxes and fall back cleanly to whole-block boxes when needed.
