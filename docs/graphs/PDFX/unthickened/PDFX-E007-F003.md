## Short description

Add a FastAPI middleware that generates a per-request UUID and binds it to structlog context so every log line emitted during the request carries the id; enforce a content-redaction rule via unit-tested log-record inspection that prevents raw PDF bytes, extracted field values, and full model prompts from ever landing in logs.
