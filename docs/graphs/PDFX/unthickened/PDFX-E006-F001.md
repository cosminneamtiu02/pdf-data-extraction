## Short description

Define the public API's Pydantic models in `features/extraction/schemas/`, one class per file: `ExtractRequest`, `ExtractResponse`, `ExtractedField`, `BoundingBoxRef`, `ExtractionMetadata`, `OutputMode` (enum), `FieldStatus` (enum). Fully type-annotated, `pyright --strict` clean.
