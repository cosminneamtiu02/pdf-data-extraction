## Short description

Add `SKILL_NOT_FOUND` (404) and `SKILL_VALIDATION_FAILED` (startup-only, non-zero exit) to `errors.yaml` with codegen and translations; raise `SKILL_NOT_FOUND` from the runtime resolution path on unknown `(name, version)` pairs or `latest` against an empty skill name.
